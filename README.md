# Análise Semântica e Geração de Código

Projeto acadêmico com foco na implementação da fase de análise semântica e geração de código de um compilador simples.

## Objetivo
Implementar mecanismos de verificação semântica, gerenciamento de tabela de símbolos e geração de código intermediário, reforçando conceitos fundamentais de Compiladores.

## Funcionalidades
- Construção e validação da tabela de símbolos
- Verificação de tipos e escopo
- Detecção de erros semânticos
- Geração de código a partir das estruturas analisadas

## Tecnologias
- Python
- Jupyter Notebook

## Contexto
Projeto desenvolvido no contexto da disciplina de Compiladores, com implementação autoral e foco didático.
